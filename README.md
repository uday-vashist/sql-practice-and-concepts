# SQL Practice and Concepts

This repository tracks my ongoing SQL learning and hands-on practice.

I regularly add queries based on problems I solve and concepts I learn, to steadily improve my SQL problem-solving skills.

## Repository Structure

### Practice-Problems
SQL solutions to practice problems, organized by difficulty level:

- Easy – Basic filtering and foundational query logic  
- Medium – Intermediate queries involving joins, subqueries, and aggregations  
- Hard – Advanced SQL challenges (to be added as I progress)

### Concepts
Small, focused queries written to understand specific SQL concepts such as joins, subqueries, window functions, grouping, and NULL handling.

### Notes
Key SQL notes and important learnings gathered during practice.
